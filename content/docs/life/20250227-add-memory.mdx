## 32GB → 96GB!

ChromeでYouTubeを見ながら作業をしていると、メモリが不足することが多々ありました。
特に最近はAndroid開発の勉強をしており、Android Studioでエミュレータを立ち上げるとメモリが逼迫してしまうのが悩みの種でした。  
(PCを起動するだけで10GB以上のメモリを消費するため、非常に辛かったです。)

購入したのは、Crucial PRO DDR5-5600 32GBX2枚です。
マザーボードはMSIのPRO Z690-A WIFIを使用しているため、本当は192GBまで増設可能ですが、  
今回は32x2を購入して現在のメモリと合わせて96GBにしました。（お金がないので...）

現在は、メモリの増設によって、より快適に作業を行うことができています。これからも効率的に作業を進めていきたいと思います🫡
